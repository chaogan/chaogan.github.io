Linux主机Apache自动跳转HTTPS


2017-4-28

耶，网上大神多非常多。
自从装了老薛主机，这几天就在折腾博客了。发现自己十分小白，并且需要学习的地方真的是非常的多，入正题。
偶尔发现输入jd.com后，浏览器会自动跳转到https://www.jd.com/觉得很神奇，于是输入自己的地址，发现不会自动跳转到HTTPS呢，觉得不解，谷歌百度查查查，也问了空间商，就丢了一个什么htaccess 规则，再问也不说了，对于我这些小白无疑是打击性的，htaccess规则很深奥，看不懂。
然后换个关键词搜索，如http自动跳转https，出来很多也是不懂额代码，观察了很多，也对比很多，最后把网站备份，动手。
我是这么操作的，
找到网站根目录下.htaccess文件，下载，然后添加
RewriteCond %{SERVER_PORT} 80
RewriteRule ^(.*)$ https://%{HTTP_HOST}/$1 [R,L]
然后保存、上传，刷新。发现OK了，自动跳转HTTPS了，哈哈，把我乐，可惜没人分享。
